# Healthsites Campaign Site 2026
[MIT License](LICENSE) © 2026 healthsites.io

A vanilla HTML/CSS/JS site for the Healthsites.io 2026 campaign. Contributions welcome!

Local Development
Run the site on your localhost easily.

Clone the repo:

bash
git clone git@github.com:healthsites/campaign-site-2026.git
cd campaign-site-2026
Open index.html in your browser:

Drag to Chrome/Firefox, or

Python server: python -m http.server 8000 → http://localhost:8000, or

Node: npx serve . → http://localhost:3000

No build tools or dependencies required—pure vanilla!

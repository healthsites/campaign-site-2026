{% if is_fr %}« Un bien commun de données n'est pas une technologie, c'est une forme de gouvernance. » {% elif is_pt %}“Um bem comum de dados não é uma tecnologia, é uma forma de governança.” {% elif is_ar %}«المشاع البياني ليس تقنية، بل هو شكل من أشكال الحوكمة.» {% else %}“A data commons is not a technology, it is a form of governance.”{% endif %}

{% if is_fr %}healthsites.io construit un bien commun de données ouvertes sur les établissements de santé avec OpenStreetMap. Les données sur les établissements de santé, comme l'eau potable ou l'air pur, constituent un bien commun — et leur gouvernance doit le refléter. {% elif is_pt %}A healthsites.io está a construir um bem comum de dados abertos sobre estabelecimentos de saúde com o OpenStreetMap. Os dados sobre estabelecimentos de saúde, como a água potável ou o ar puro, funcionam como um bem comum — e a sua governança deve refletir isso. {% elif is_ar %}تبني healthsites.io مشاعًا مفتوحًا لبيانات المرافق الصحية مع OpenStreetMap. تعمل بيانات المرافق الصحية، مثل المياه النظيفة أو الهواء النقي، كمنفعة عامة — وينبغي أن تعكس حوكمتها ذلك. {% else %}healthsites.io is building an open data commons of health facility data with OpenStreetMap. Health facility data, like clean water or fresh air, functions as a common good — and its governance should reflect that.{% endif %}

{% if is_fr %}Pour les ministères de la Santé, cela reformule l'argument à présenter à leurs ministères des Finances : les données validées sur les établissements de santé ne sont pas un coût de projet récurrent, mais un actif d'infrastructure nationale entretenu en permanence. {% elif is_pt %}Para os Ministérios da Saúde, isto reformula o argumento a apresentar aos seus Ministérios das Finanças: os dados validados sobre estabelecimentos de saúde não são um custo recorrente de projeto, mas um ativo de infraestrutura nacional permanentemente mantido. {% elif is_ar %}بالنسبة لوزارات الصحة، يعيد هذا صياغة الحجة التي تُقدَّم إلى وزارات المالية: بيانات المرافق الصحية المتحقق منها ليست تكلفة مشروع متكررة، بل أصل بنية تحتية وطنية يُصان بشكل دائم. {% else %}For Ministries of Health, this reframes the case to bring to their Ministries of Finance: validated health facility data is not a recurring project cost, but a permanently maintained national infrastructure asset.{% endif %}
